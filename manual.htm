1. Overview

Frameshop is a practical WYSIWYG tool for game developers to create texture atlases and frame animations. Considering texture and animation packaging is all along important for game development on each platform, an efficient tool is always profoundly required, Frameshop can meet these requirements well. It runs on Windows only for the moment.

2. Main Features

  • Texture atlas packaging;

  • Multi atlas page support;

  • Atlas frame rotation support;

  • Atlas frame padding support;

  • Format conversion plugin system;

  • Frame animation editing;

  • Animation helper rectangle data editing.

3. Usages

3.1 Main Window User Interface

The main window appears as below after you started Frameshop; it is composited by five areas.

3.1.1 Toolbar

Toolbar sits at the top, most of the project/texture file operations are done here.

3.1.2 Page View

There is a big "Page" view in the middle of the main window, we can create/remove texture atlas pages and add/remove frames under a page here.

The checkered picture box shows the content of a page. A page is represented as a folder in a tree view at the left top of the page view; all frames associated in a page are listed under a page folder representation. Page command strip is at the bottom.

3.1.3 Inspector View

Inspector view is as known as property panel in some other software, all editable properties of a page can be modified with it.

3.1.4 Frame List

Texture assets frames are listed here in preview mode, you can add new ones to the list or remove from it, or pick textures to add to an atlas page.

3.1.5 Output Panel

Runtime status information is shown in the box.

3.2 Creating a New Package

Click the "New" button on the toolbar to create a Frameshop project package, a configuration window will show as below.

Choose an appropriate resolution for a texture atlas page, after that new created pages will use that specified size as default; the size of each page can be changed independently later.

Vector texture scale is a factor that will be applied to a .SVG (Scalable Vector Graphics) file at the importing moment.

3.3 Importing Frames

Click the "Add Texture" button on the toolbar to add texture files into working project as assets, or click the "Add Folder" button to add textures inside a folder (recursively as default). Imported textures will be listed in the frame list. You can also drag and drop a texture file or folder to the frame list to do so.

3.3.1 Supported Formats

Frameshop supports five types of texture file as below.

  • PNG

  • BMP

  • JPG

  • SVG

  • TGA

3.4 Adding Frames to a Page

To add frame assets into an atlas page, select a target page in the page/frame list to add to, then simply drag them from the frame list and drop to the center page view area; frames can be multi selected while Ctrl key is pressed down. The page/frame list and the page atlas view will be updated immediately when a frame is added. You can double click on a frame assets to perform adding as well.

Frameshop neither copies original textures to the project folder nor parse them into project file, it reads original raw texture files each time you open a project. Frameshop saves a relative path if a texture file is under the same disk with a project file; otherwise it saves an absolute path.

3.5 Saving a Package

Click the "Save" button to save a Frameshop project package to disk.

Click the "Save As" button to save current Frameshop project to another path and use that path as the working one.

3.6 Opening a Package

Click the "Open" button to show an open file dialog, and then choose a Frameshop project file to load it. A project file is typed as .FSP; Frameshop can relate to it so that Frameshop could be launched automatically when you tried to open a project file directly. You can also drag and drop a project file to the toolbar to do so.

3.7 Creating a New Page

Click the "+" button below the page/frame list in the page view panel to add a new page.

3.8 Removing a Page

Click the "-" button below the page atlas view to remove current page.

3.9 Removing Frames from a Page

Double click a frame in the page atlas view to remove it. Or right click on a frame item (either in the page/frame list or in the page atlas view) and choose "Remove from Page" to do so.

3.10 Searching a Texture

Pick a single item from the combo box on top of the frame list to show it only in the frame list view. Or type some letters and hit enter to match and filter frames approximately.

3.11 Editing Page Properties

In the inspector view we can edit properties of a selected page.

A page name is unique in a Frameshop project scope; and it CANNOT contains any of these charactors: , . ! ? * : | < > " / \.

If the "AllowRotation" property is set true, Frameshop might try to rotate a frame when packing it into an atlas to optimize space occupation.

The "Padding" value will be applied to each frame in a page to make them get a few pixels away from other ones.

The "TextureSize" specifies a size of current page.

Modifications of "AllowRotation", "Padding" and "TextureSize" may cause atlas relayouting, the page may not be able to hold all packed frames after that relayouting, if so an ensurance dialog box will appear to ask you if you'd like to revert the change or keep it with some frame abandon in current atlas page.

3.12 Publishing

Click the "Publish" button to export current Frameshop project, and make essential conversions at the same folder where the project file stays.

3.13 Format Conversion Plugins

Frameshop supports customized command line tool invoking when publishing a project, so that we can make some format conversions with this functionality.

See 3.21.1 to get detail information about how to customize a conversion plugin.

3.14 Animation Window User Interface

Click the "Animation" button to open the animation editing window which appears as below.

The left side is a page/frame list which shows same contents as the one in the main window. The middle view is frame preview area. The right side is composited by a sequence list on top and a frame (in sequence) property grid.

3.15 Creating a New Animation

Choose "File" -> "New" to create a new animation sequence.

3.16 Adding Frames to an Animation Sequence

Double click a frame item in the page/frame list to add it to the sequence. Or right click on an item and choose "Add to Sequence" to do so as well.

3.17 Modifying Frame Offset in an Animation

Click on a frame in the sequence to select it.

Choose "Edit" -> "Move Mode" to active move mode; then drag the frame in the preview area. Or in another way edit the "Offset" property in the property grid to do a fine adjustment.

3.18 Modifying Frame Duration in an Animation

Click on a frame in the sequence to select it.

Edit the "Time" property in the property grid.

3.19 Preview an Animation

Choose "Play" -> "Preview" to open the preview window as below.

You must be familiar with this easy to use player style window. Click the triangle button to play, double parallel line button to pause, square button to stop, clip button to switch loop on/off; use the slider bar on the timeline to pick frames.

3.20 Animation Helper Rectangles

Each frame in a sequence can contains three helper rectangles, these rectangles are independent data, you can use them for specific usage such as collision detection or object binding in your engine.

To edit a helper rectangle, follow the steps below.

Click on a frame in the sequence to select it.

Choose "Edit" -> "Quad Mode N" to active quad mode; then drag out a rectangle. Or in another way edit the "RectN" property in the property grid to do a fine adjustment. Notice 1, 2 and 3 substituteÂ for "N" here.

3.21 Options

Click the "Option" button on the main window to open the option configuration window. There are two tabs as below.

3.21.1 Post Events

Format conversion plugin is a text based file that similar to a system shell processing batch; it doesn't do anything itself, but invokes other command line tools and pass arguments to them.

Frameshop can use four kinds of plugins that respectively process project folder, each texture, each page data, each animation. Plugins are all located in the "plugin" folder under the same folder where the Frameshop main EXE sits, a plugin is a .FSC file that you can view and edit using any text editor.

All plugins are listed in each combo box associated to operation types. You can simply choose an exist one to use it to process assuming targets; type an desired new plugin name in the combo box and press the "Edit" button to create a new plugin file.

The plugin syntax is as simple as a single line parameterized command with escape lexical tokens. Use the escape tokens below to represent published files/folders.

  • $FULL$: full path

  • $PARENT$: parent folder

  • $FILE$: file name only

  • $EXT$: file extension only

3.21.2 Relate Frameshop Project File Type

Open the options configuration window, switch to the "Misc" tab.

Check/uncheck "Relate Frameshop project (*.fsp) files" to relate/unrelate project files to Frameshop. You can double click a .FSP file to open it after it's related.

3.21.3 Ensure Before Removing a Frame

Open the options configuration window, switch to the "Misc" tab.

Check "Ensure before removing a frame" to show an insurance dialog after you were tending to remove a frame from the assets list.

3.21.4 Add Texture Folders Recursively

Open the options configuration window, switch to the "Misc" tab.

Check "Add texture folders recursively" to parse all sub folders when parsing an outer one.
